# Chat em Tempo Real

## Sobre o projeto:

Um chat em tempo real desenvolvido com **Node.js**, **Express**, **Socket.io** e **React**, permitindo:

- Login por nickname
- Chat global
- Chats privados (salas simples)
- Sussurros
- Lista de usuários online
- Atualização em tempo real

---

## Tecnologias utilizadas:

### Backend

- Node.js
- Express
- Socket.io

### Frontend

- React
- Styled-components
- Socket.io-client

---

## Como rodar o projeto?

### 1. Clone o repositório

`git clone https://github.com/lidnigth/desafio-chatglobal.git`
`code .`

### 2. No backend:

acesse a pasta:
`cd backend`
instale dependências:
`npm install`
inicie o servidor:
`npm start` ou `npm run dev`

### 3. No frontend:

acesse a pasta:
`cd frontend`
instale dependências:
`npm install`
inicie o app:
`npm start` ou `npm run dev`

## Testando:

Abra uma aba em navegadores diferentes, use nicknames diferentes e envie mensagens ou, se estiver no chat global, sussurros para conferir o funcionamente. (Se envia sussuro com @ + nickname do usuario que deseja).
